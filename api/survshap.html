survinsights.local_explaination.survshap

Computes SurvSHAP for the survival model.

Parameters:

  • explainer (Python object, required): Instance of the explainer class for the survival model.
  • new_data (pd.DataFrame, required): Data for new observations whose predictions need to be explained.
  • sample_id (int, optional, default: 0): Index of the target observation in data.

Returns: DataFrame containing SurvSHAP values for each feature.
