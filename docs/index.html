Survival Model Explainer

Enhance the interpretation of survival models with personalized insights and tools.

About the Package

The Survival Model Explainer package is a Python tool designed to simplify the interpretation of survival models. It helps users:

  • Explain survival predictions with methods like SHAP and ICE.
  • Identify the most suitable interpretation methods for specific datasets.
  • Analyze longitudinal data and dynamic time-dependent survival models.

Installation

pip install survival-model-explainer

Getting Started

Here's a quick example of how to use the package:


from survival_explainer import explain_model

# Load your survival model and dataset
model = ...
data = ...

# Get explanations
explanations = explain_model(model, data)
