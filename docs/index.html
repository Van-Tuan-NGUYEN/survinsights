survinsights

Enhance the interpretation of survival models with personalized insights and tools.

About the Package

The survinsights package is a Python tool designed to simplify the interpretation of survival models. It helps users:

  • Explain survival predictions with model-agnostic methods (SurvSHAP, SurvLIME, PFI, etc).
  • Identify the most suitable interpretation methods for specific datasets.
  • Analyze longitudinal data and dynamic time-dependent survival models.

Installation

pip install survinsights

Getting Started

Here's a quick example of how to use the package:


from survinsights import explainer

# Load your survival model and dataset
model = ...
data = ...

# Get explanations
explanations = model_explainer(model, data)
